Ambience
===========

An animated gradient plugin for jQuery.


Examples
--------------

- [Basic](http://christophercliff.github.com/ambience/ "Basic")


Requirements
-------------

- jQuery 1.4.2
- jQuery UI 1.8.5


Usage
----------------

The HTML:

    <div class="myambience"></div>

The JavaScript:

    $('.myambience')
        .ambience()
        ;
